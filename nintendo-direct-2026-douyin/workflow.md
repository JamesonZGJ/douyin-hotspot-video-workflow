# Nintendo Direct 6.10.2026 多游戏总结制作记录

## 流程

| 步骤 | 处理 |
|---|---|
| 1 | 用 Nintendo 官方回顾页核对游戏名单和日期口径 |
| 2 | 使用用户下载的整场发布会视频 `assets/hd/source.mp4` |
| 3 | 从整场视频抽 contact sheet，挑出可用动态片段 |
| 4 | 裁出 17 段横版 H.264 片段，放到 `assets/clips-v2/` |
| 5 | 重写旁白成约 2 分 39 秒多游戏总结，不再压到 1 分钟 |
| 6 | 参考玩游戏的阿舔的节奏重写：先爆点，再判断，再用玩家视角收束 |
| 7 | 保留之前 State of Play / Xbox 快报的红黑竖屏包装 |
| 8 | 开头改成《塞尔达传说：时之笛》重制，不再泛泛铺垫直面会 |
| 9 | 用 `zh-CN-YunxiNeural` 生成年轻男声旁白 |
| 10 | 跑 HyperFrames `lint / validate / inspect / render` |
| 11 | 用 ffprobe 检查最终视频流、音频流、尺寸和时长 |

## 素材原则

| 原则 | 说明 |
|---|---|
| 主素材 | 只用用户给的完整发布会视频 |
| 声音 | 原视频无音频流，所以成片声音来自新版旁白 |
| 事实 | 游戏名、时间和平台口径以 Nintendo 官方页面为准 |
| 游戏名 | 口播用中文和玩家熟悉叫法；台港官方写法保留在数据文件里 |
| 版式 | 横版发布会画面放在竖屏中部，不硬铺满 |
| 节奏 | 多游戏总结优先信息密度，不强行做 60 秒短快报 |
