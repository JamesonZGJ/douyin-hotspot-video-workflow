# 热点发布会短视频固定模板

用途：把一场发布会、直播、发布文章，做成 55 到 65 秒的抖音竖屏快报。

固定成片规格：

- 画布：1080x1920。
- 主画面：中部 3:2 游戏画面框，素材完整展示，不铺满全屏。
- 字幕：底部大字幕，不遮挡主画面。
- 旁白：中文口播优先，游戏名必须中文读出。
- 素材：官方视频能用就用，优先 1080p；片头、logo、黑场、年龄门过不去时，用官方截图。
- 同步：时间轴必须按旁白分段走，不能按素材长度硬切。

## 复用步骤

1. 复制这个模板目录，改成新项目名。
2. 填 `facts.template.json`：游戏名、日期、重点、来源、素材。
3. 写 `narration.template.zh.txt`：控制在 8 到 12 句。
4. 用 `timeline.template.json` 拆每一句的起止时间。
5. 把 `index.html` 里的画面起止时间、字幕起止时间同步成同一套时间。
6. 跑检查：

```powershell
npm run lint
npm run validate
npm run inspect -- --samples 14
```

7. 渲染：

```powershell
npm run render -- --fps 30 --quality high --output renders\final.mp4
```

## 硬性检查

- 每个游戏画面必须覆盖它被旁白提到的完整时间。
- 不能在旁白还讲 A 游戏时切到 B 游戏。
- 3:2 画面框里不能只有 logo、黑场、片头文字。
- 动态素材低于 1080p 时，先重下高清源，再渲染成片。
- 画面如果是截图，必须来自官方文章或官方视频抽帧。
- 最终时长控制在 55 到 65 秒。
